# WOM-PROJEKT

## Setup

`npm install`

- Öppna 2 nya terminaler:

  - `tsc --watch` För typescript building
  - `npm run dev` För express APIn

  ***

- Gör en .env fil
  - Laga med:
    - `DATABASE_URL` till mongodb atlas
    - `PORT` för APIn

---

APIns funktionalitet kan testas med `API/REST-client/DBRequests.http`
